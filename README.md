# Breezy Weather Web App (微風天氣)

A premium, modern weather web application inspired by "Breezy Weather", built with React, Vite, and Material 3 design principles.

一個高品質、現代化的天氣網頁應用程式，深受「Breezy Weather」設計啟發，使用 React、Vite 打造，並遵循 Material 3 設計規範。

---

## 🌟 Features (功能重點)

- **🎨 Modern Aesthetics (現代美學)**: Material 3 inspired UI with smooth transitions and rounded corners. (遵循 Material 3 的圓角與極簡設計，具備流暢過場效果。)
- **🌍 Multi-language Support (多語系支援)**: Supports English, Traditional Chinese, Simplified Chinese, and Japanese. (支援英文、繁體中文、簡體中文與日文。)
- **📍 Smart Geolocation (智慧定位)**: Auto-detects your location via IP and provides a powerful search with up to 5 location suggestions. (透過 IP 自動偵測所在地，並提供支援五個候選地點的地點搜尋功能。)
- **📈 Comprehensive Forecast (全面預報)**:
  - 48-hour hourly forecast with precipitation probability. (48 小時逐時預報，包含詳細降雨機率。)
  - 7-day daily forecast. (未來七天天氣預報。)
- **🖱️ Intuitive Navigation (直覺的操作)**: Supports mouse drag-to-scroll and buttons for hourly data. (支援滑鼠拖拉、滾輪與按鈕導覽每小時預報。)
- **🌈 Dynamic Backgrounds (動態情境背景)**: The UI background updates themed linear gradients based on the current weather condition (Clear, Cloudy, Rain, Snow, etc.). (背景顏色會根據當前天氣狀況——晴天、陰天、雨天、降雪等——自動切換動態漸層色彩。)
- **🕒 Real-time Data (即時資訊)**: "Last Updated" timestamp to ensure data freshness. (顯示資料「更新時間」，確保導覽最新氣象資訊。)

---

## 🛠️ Technology Stack (技術棧)

- **Frontend**: React.js (Hooks, Functional Components)
- **Tooling**: Vite (Fast HMR and bundling)
- **Icons**: Lucide React
- **API**: [Open-Meteo](https://open-meteo.com/) (Weather & Geocoding)
- **Styling**: Vanilla CSS (CSS Variables, Flexbox/Grid)

---

## 🚀 Getting Started (本地運行)

1. **Clone the repository (複製儲存庫)**:
   ```bash
   git clone https://github.com/sos19941015/breezy-web.git
   cd breezy-web
   ```

2. **Install dependencies (安裝依賴)**:
   ```bash
   npm install
   ```

3. **Run development server (啟動開發伺服器)**:
   ```bash
   npm run dev
   ```
